
# Helper Commands

## Browser

**Stop all Edge instances** — Force-kills every running Edge process and its children.
```bat
taskkill /F /IM msedge.exe /T
```

**Open remote debug endpoint** — Inspect active pages when Edge is launched with `--remote-debugging-port=9222`.
```
http://localhost:9222/json
```

---

## Server

**Kill process on port 8420** — Finds the PID listening on 8420 and force-terminates it.
```bat
for /f "tokens=5" %a in ('netstat -aon ^| findstr :8420') do taskkill /f /pid %a
```

---

## Dev Server

**Start dashboard** — Clears any stale DB lock file, then starts the dev server with the correct DB path.
```bash
cd c:/dev/garden-manager/dashboard
rm -rf ../data/garden.db.lock; DB_PATH=../data/garden.db npm run dev
```

---

## Database

**Reset the database** — Deletes the current database and lock file, then re-initialises the schema.
```bash
rm -rf /c/dev/garden-manager/data/garden.db
rm -rf /c/dev/garden-manager/data/garden.db.lock
node ../db/init-db.js
```
